The Orion Email Customizer for Woocommerce plugin lets you customize your woocommerce emails  with an intuitive template builder. The plugin allows you to add different elements in the template like your social media link, logo, and a lot more. It makes your template precise and easy to understand with a layout that matches your brand style.


<h3>FREE VERSION FEATURES </h3>

<ul>
 <li>
 <h4>DEFAULT TEMPLATES </h4>
 The plugin comes with 5 default WooCommerce email templates that can be used for all email types. These templates can be customized using the customization options built into the plugin.
 </li>
 <li>
 <h4>USER-FRIENDLY</h4>
 This easy-to-build email template builder provides the best user experience by helping users customize & manage the WooCommerce templates effortlessly.
 </li>

== Installation ==

<div class="block-content"><h4>Minimum Requirements</h4>

<ul>
<li>WordPress 5.0 or greater</li>
<li>Woocommerce 5.0 or greater</li>
<li>PHP version 7.0 or greater</li>
<li>MySQL version 5.0 or greater</li>
</ul>

<strong>This section describes how to install the plugin and get it working</strong>

<h4>Automatic installation (easiest way)</h4>

To do an automatic install of ECWO, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "ECWO" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".

<h4>Manual installation</h4>

<strong>Uploading in WordPress Dashboard</strong>

<ol>
<li>Download <code>orion-email-customizer-for-woocommerce.zip</code></li>
<li>Navigate to the 'Add New' in the plugins dashboard</li>
<li>Navigate to the 'Upload' area</li>
<li>Select <code>orion-email-customizer-for-woocommerce.zip</code> from your computer</li>
<li>Click 'Install Now'</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

The WordPress codex contains <a href="https://wordpress.org/support/article/managing-plugins/#installing-plugins" rel="nofollow">instructions on how to install a WordPress plugin</a>.

<h4>Updating</h4>

You can use the automatic update feature of WordPress to update the plugin safely.</div>
